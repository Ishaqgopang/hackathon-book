# Physical AI & Humanoid Robotics Course

Welcome to the comprehensive course on building intelligent humanoid robots using modern robotics technologies.

## Course Overview

This course is designed to take you from beginner to advanced in the field of Physical AI and humanoid robotics. You'll learn about:

- **ROS 2**: The middleware that connects AI algorithms to physical robot hardware
- **Digital Twins**: Physics-based simulation using Gazebo and high-fidelity visualization with Unity
- **NVIDIA Isaac**: AI-powered perception and navigation for intelligent robots
- **Vision-Language-Action**: Building autonomous humanoid systems

## Course Modules
